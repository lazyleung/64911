This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Controller files

  • CarButtonControl.java -- Implements the Car button controller
  • CarPositionControl.java-- implements the Car position controller
  • Dispatcher.java-- implements the dispatcher
  • DoorControl.java-- implements the door controller
  • DriveControl.java-- implements the drive controller
  • HallButtonControl.java-- implements the hall button controller
  • LanternControl.java-- implements the lantern controller

Translator files

Links and descriptions to all message translators used by the controllers.
  • BooleanCanTranslator.java -- translates boolean can message payloads
  • CarCallCanPayloadTranslator.java-- translates car call can message payloads
  • DesiredDwellCanPayloadTranslator.java-- translates desired dewell can message payloads
  • DoorMotorCanPayloadTranslator.java-- translates door motor can message payloads
  • DriveSpeedCanPayloadTranslator.java-- translates drive speed can message payloads
  • HallCallCanPayloadTranslator.java-- translates hall call can message payloads

Utility files

  • Utility.java - contains various utility functions

Monitor files

  • Proj7RuntimeMonitor.java - Monitors the number of overweights, number of wasted door openings, and total time used for door reversal update the filename link
